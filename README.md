# 🏠 Real Estate Market Analysis Using Python

## 📌 Problem Statement
To analyze real estate market data and identify factors affecting property prices and demand trends.

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Workflow
- Data Cleaning (Removed duplicates & handled missing values)
- Feature Engineering (Price per sq.ft calculation)
- Exploratory Data Analysis (EDA)
- Visualization using scatter plots, bar charts & heatmaps

## 📈 Key Insights
- Locality significantly affects property prices
- Apartments show higher demand
- Approval status impacts pricing
- BHK count has limited price correlation
